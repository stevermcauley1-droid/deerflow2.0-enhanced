# Jobs2Go Scripts

This directory is reserved for small utility scripts used during pilot delivery.

Examples for later steps:
- migration validation helpers
- OpenAPI lint helpers
- payload schema checks

Current scripts:
- `run_local_launch_gates.ps1` - one-command local readiness gate for Windows PowerShell.
- `smoke_jobs2go_endpoints.ps1` - launch-day smoke checks for health, ops APIs, and UI routes.
- `validate_openapi_examples.py` - validates JSON parseability of OpenAPI example payloads.
