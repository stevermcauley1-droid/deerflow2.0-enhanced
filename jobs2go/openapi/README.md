# Jobs2Go OpenAPI Package

This package provides domain-split API contracts for the Jobs2Go pilot.

## Domains
- `domains/auth.yaml`
- `domains/workers.yaml`
- `domains/jobs-matching.yaml`
- `domains/offers-bookings.yaml`
- `domains/payments.yaml`
- `domains/chat-reviews-trust.yaml`

## Manifest
- `index.yaml` lists all domain specs and intended ownership.

## Examples
- `examples/` contains request/response JSON payloads for contract testing and frontend integration.
